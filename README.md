# Game
A simple game, inspired by Vito and Sorin.

## Board
> Like a [push pop bubble fidget](https://duckduckgo.com/?q=push+pop+fidget&t=osx&ia=images&iax=images)
- Columns with n dots.
- n = 3, 5, 6, 6, 5, 3.

## Rules
- Pick a column.
- Push some dots.
- Pass board to peer.

## Loses
- Last dot loses.
- => 1 dot left, you win.
- => 0 dots left, you lose.
